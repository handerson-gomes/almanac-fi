# 054a — Debt payoff and investing comparison feature

## Story

As a user, I can compare debt-payoff and investing allocation strategies using explicit assumptions and shared future cash flow.

## Requirements

Create a feature with deterministic payoff/amortization, interest-cost, contribution-opportunity, and scenario-comparison calculations; add assumption workflow, dashboard, capability/MCP contributions, allocation proposal integration, methodology citations, fixtures, and evaluations. Present results as scenarios, not personalized financial advice.

## Acceptance criteria

All rates, returns, taxes, and contribution assumptions are visible; debt payments and allocations use the shared forecast/ledger; the feature detects conflict with existing goals; outputs cite methodology versions and missing information.
