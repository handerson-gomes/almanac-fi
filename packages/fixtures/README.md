# Synthetic household fixtures

- `complete-household`: income, transfers, budget, dependent, goal, debt, investment, and an explicit missing profile fact.
- `missing-data`: intentionally incomplete account and goal data for validation and assistant prompting.
- `import-correction`: an original and corrected import record for reconciliation tests.

All names, institutions, identifiers, and amounts are deterministic synthetic data. They are not derived from real people or accounts.
